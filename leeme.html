Proyecto

Aplicación Web para el intercambio de cartas Magic: The Gathering, en la cual, cualquier usuario previamente registrado podrá subir sus cartas intercambiables y buscar cartas de otros usuarios con la posibilidad de proponerles un intercambio. Cada usuario podrá consultar los intercambios que le han propuesto, para aceptarlos, rechazarlos o realizar una contraoferta al usuario que le ha propuesto el cambio. En caso de aceptarlos, la aplicación compartirá los datos de contacto entre los usuarios involucrados en el cambio, para que estos realicen físicamente el cambio de la forma que consideren (presencial, envío por correo, etc). Despues de aceptar el intercambio, ambos deben confirmar que ha tenido lugar. Una vez finalizado un intercambio no se podrá modificar. Los usuarios podrán valorar el trato del otro usuario solo una vez después de un intercambio finalizado. La información sobre la valoración de cada usuario podrá ser consultada antes de realizar un intercambio. Los usuarios podrán ser coleccionistas, con todas las funcionalidades anteriormente citadas, o administradores, que podrán eliminar usuarios o actualizar la base de datos de cartas.

Todas las modificaciones pedidas en el examen se han realizado de manera satisfactoria:
  • En negrita aparecen las cartas que buscas y que poseen el resto de usuarios, y viceversa
  • Acceso a perfil de los usuarios en el home
  • Mostrar un mensaje en el home cuando tiene el usuario ofertas pendientes
  • Mostrar el manaCost con 'bolas'
También, la obtención de usuarios/cartas se realiza de forma 'fresco', por lo que hemos suprimido bastantes merge, persist, etc Por último, hemos modificado los setColeccion por addElement o deleteElement, como nos recomendaste.

Vistas

  • Pantalla de inicio -Página de inicio con capacidad para loguearse y registrarse en la aplicación web
  • Buscador de cartas - Página principal con acceso a la lista de usuarios, mostrándonos sus cartas intercambiables y las cartas que necesita cada usuario. Pudiendo filtrar los usuarios mostrados
  • Pantalla del Administrador - Contiene los elementos necesarios para la gestión de usuarios y de las colecciones; y a la que se accede tras el login en la vista de "Login Administrador"
  • Perfil de usuario - Muestra el perfil del usuario con sus datos, sus valoraciones y la posibilidad de modificar los datos personales
  • Gestión de tus cartas - Gestión de la colección de cartas del usuario.
  • Gestión del intercambio - Muestra la interfaz para que el usuario de la aplicación pueda intercambiar sus cartas Magic con otro usuario previamente seleccionado.
  • Contraoferta - Vista para realziar una contraoferta. Solo accesible si tienes un intercambio en estado 'Pendiente'
  • Historial de intercambios - Muestra tus ofertas y los de otros usuarios. El usuario puede filtrar ofertas y cambiar el estado de una oferta.
  • Información página web - Muestra información sobre la página.
  • Información de los términos y condiciones .
